# HSL Traffic Map

This repository contains a simple React application that displays real-time traffic data on a map using the Leaflet library. The application subscribes to MQTT topics to receive vehicle position updates and visualizes them on the map.

The data is sourced from the Helsinki Regional Transport Authority (HSL) and is licensed under the Creative Commons name 4.0 (CC BY) license: https://digitransit.fi/en/developers/apis/7-terms-of-use/.
