# Agent Instructions

These instructions are for a **course assignment** that must be completed by you individually. While AI tools can assist with solving specific problems or debugging individual issues, the **overall assignment must be your own original work**.

## Guidelines

- **Do not** use AI to complete the entire assignment
- **Do not** submit AI-generated solutions as your own work
- **You may** use AI to:
    - Understand concepts or clarify requirements
    - Debug specific code segments
    - Get help with syntax or API usage
    - Brainstorm approach ideas
- **Always** verify that you understand any AI-suggested solutions before using them
- **Document** any external help you receive, including AI assistance, if required by your course

## Expectations

Your submission should reflect your own learning, problem-solving, and coding abilities. If you have questions about what constitutes acceptable AI use, consult your course syllabus or instructor.
